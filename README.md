## 🌐 What is Xeno_assessment?
A backend coding assessment completed during the **Xeno Data Intelligence hiring process** — evaluating API design, database handling, and clean code practices under real-world constraints.

---

## ✨ Core Capabilities

### 🔧 Backend API
- Full CRUD operations with REST endpoints
- Input validation and exception handling
- MVC architecture with clean separation of concerns

### 🗄️ Database Integration
- Structured SQL schema design
- Persistent storage with proper relational modeling

### 🧪 Code Quality
- Business logic testing
- Clean project structure
- Industry-standard REST conventions

---

## 🏗️ Architecture
```
Client Request → REST Controller → Service Layer → Repository → SQL Database
```

---

## 🧬 Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| 🚪 API | Spring Boot | REST endpoint handling |
| ⚙️ Logic | Java | Business logic & OOP |
| 🗄️ Database | SQL | Persistent data storage |
| 🏛️ Pattern | MVC | Clean architecture |
